# Block-Element-Escape

文字を16進数バイナリへと変換し、それをブロック要素に置換することで、まるで暗号のような文字を作り出すツールです。

## Features
- **変換**: テキストをブロック文字へと置換します。
- **復元**: ブロック文字から元の文字列を復元します。
- **反復処理**: カウントを増やすことで、複数回の変換が可能です。

## Usage
1. `input` フィールドに「変換/復元」したい文字を入力します。
2. `Count` 数字を調整することで、複数回の変換・復元ができます。
   - ※クラッシュ対策のため、最大5までに制限しています。
3. `Mode` を切り替えることで、エンコードとデコードを選択できます。

## Disclaimer
本ツールは反復計算を行うごとに、文字数が指数関数的に増加します（1文字につき $4^{n}$ 倍に増殖）。そのため、回数を増やしすぎたり長文を入力したりすると、ブラウザがフリーズまたはクラッシュする恐れがあります。
